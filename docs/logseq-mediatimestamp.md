# logseq-mediatimestamp-plugin

# Overview

Inserts and navigates to timestamps for locally embedded media files.

# Usage

1. Upload a media file.
2. Create a child block under the block containing the media file.
3. Type `/Insert timestamp`. Continue adding description of timestamp.
4. Click to on the button to navigate to the specific timestamp.

# Installation

Recommend to install from the marketplace (if available). If not, download a release and manually load it in Logseq.
